Sourcerer 0.19.84 — First release

Instant cross-platform search across four lenses — filename, full-text content, audio attributes, and fuzzy name — fed by a realtime filesystem journal.

Phase 12 May 2026

Settings, daemon IPC & i18n

A 26-panel Settings dialog, real indexd RPC, a WASM custom-extractor sandbox, and all 18 locales wired end-to-end.

Phase 11 2026

The search UI

A Tauri + Svelte search experience with lens-grouped results, themes, and a global hotkey — the magic moment.

Phase 9 2026

Audio lens

EBU R128 loudness, peak, silence, and dynamic-range analysis with lufs:, codec:, and length: query modifiers.

Phase 8 2026

Content lens

Full-text extractors for text, PDF, Office, code in 32 languages, archives, and JSON/CSV/YAML.

Phases 1–4 2026

The index core

Per-OS filesystem journals (NTFS USN, FSEvents, fanotify) feeding a Tantivy + trigram index over a SQLite store.
